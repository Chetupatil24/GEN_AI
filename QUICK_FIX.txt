  QUICK FIX FOR BLANK PAGE / WEBSOCKET ERROR
═══════════════════════════════════════════════════════════

THE PROBLEM:
Your browser is trying to connect to ws://0.0.0.0:8503
This happens because you accessed the page via http://0.0.0.0:8503

THE FIX (3 SIMPLE STEPS):

1. CLOSE YOUR BROWSER COMPLETELY
   - Close ALL browser windows
   - Don't just close tabs

2. OPEN A NEW BROWSER WINDOW
   - Start completely fresh

3. TYPE THIS IN THE ADDRESS BAR:
   localhost:8503
   
   Then press Enter

═══════════════════════════════════════════════════════════

VERIFY IT WORKED:
1. Press F12 (Developer Tools)
2. Go to Network tab
3. Filter by "WS" (WebSocket)
4. You should see: ws://localhost:8503/_stcore/stream
5. Status should be green (101 Switching Protocols)

If you still see ws://0.0.0.0:8503, you're still using the wrong URL!
